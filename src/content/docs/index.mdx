## 这个站现在先做什么

<CardGrid stagger>
  <Card title="独立 URL" icon="open-book">
    每新增一篇教程，都生成一个稳定链接，适合放到视频、公众号、小红书和 X 里直接分发。
  </Card>
  <Card title="Markdown 管理" icon="document">
    教程正文用 Markdown 存放，方便 OpenClaw 直接创建、修改、提交和发布。
  </Card>
  <Card title="截图支持" icon="image">
    操作截图统一放在 public/images/tutorials/ 目录，教程里直接按路径引用。
  </Card>
  <Card title="自动发布" icon="setting">
    后续接 GitHub 和 Cloudflare Pages 后，可以做到写完教程即自动构建和上线。
  </Card>
